This repository contains the design and simulation of a simple Digital Filter implemented using Verilog HDL. The project was developed to understand the fundamentals of digital signal processing, synchronous digital circuits, and hardware description language programming. Digital filters are widely used in communication systems, audio processing, embedded systems, and signal processing applications to remove noise and smooth input signals.

The digital filter designed in this project demonstrates the working principle of a moving average filter. The design uses the current input sample and previous input samples to generate a filtered output. Registers are used to store previous input values, and the output is calculated by averaging multiple samples.

The project was simulated using Xilinx Vivado, and a separate testbench was written to verify the functionality of the filter. Different input values were applied during simulation, and waveform analysis was performed to observe the filtered output response.

This repository mainly contains two Verilog files:

1.digital_filter.v – Main digital filter module

2.digital_filter_tb.v – Testbench for simulation and verification

The digital filter performs the following operations:

1.Input sample processing

2.Sample averaging

3.Output smoothing

4.Synchronous data processing

5.Filtered output generation

This project helped in understanding important digital design concepts such as:

1.digital filtering

2.signal smoothing

3.synchronous sequential circuits

4.Verilog HDL syntax

5.simulation and debugging using waveforms

The design can serve as a beginner-friendly project for students learning FPGA design, Verilog HDL, digital signal processing, and embedded system concepts. Since digital filters are important in modern communication and signal processing systems, this project also acts as a foundation for advanced DSP and FPGA-based filter designs.

The project was created purely for educational and learning purposes and can be extended further by adding FIR filters, IIR filters, coefficient-based filtering, higher-order filters, and advanced DSP operations.
